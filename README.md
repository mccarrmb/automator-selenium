# automator-selenium

Ruby-based Selenium project for Google

## Requirements

* __Ruby__ >= 2.7.0
  * Bundler gem ~ 1.16
* __Operating Systems__
  * macOS >= 10.10
  * Linux (kernel >= 3.13)
  * Windows 10 >= 1706 (requires Windows Subsystem for Linux)
* __Installed Browsers:__

|       | Linux | macOS | Windows |
|:-----:|:-----:|:-----:|:-------:|
|Chrome |      X|      X|        X|
|Firefox|      X|      X|        X|
|Safari |       |      X|         |
|Edge   |       |       |        X|
